# PodcastPipe — Podcast production pipeline — track episodes, guests, and editing stages.

**Job 63** | **Package:** com.jnetai.podcastpipe

## What It Does
Podcast production pipeline — track episodes, guests, and editing stages.

## Features
- Track podcast episodes from planning to published
- Manage guests and contact details
- Track editing stages per episode
- Record recording dates and durations
- About page with update checker

## Build
- Android API 26+, Kotlin, Room, KSP, ViewBinding + XML, Material3 Dark
